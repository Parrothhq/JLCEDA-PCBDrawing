基于@sytnocui的代码，进行了部分优化，使其更适合初学者创作使用
当然，您可能需要先了解一下PCB的基本结构，这样画出来的画就更符合自己的心意:)

作画流程（主要靠自己天马行空的想象力）：
1、将图片颜色简化，太多颜色会导致合并图层很累，且没必要
2、根据源图片，取出图中的9种颜色（可再精简或增加颜色），分别填入hexColors中
3、直接执行main.py，该代码将会把所有颜色以蒙板的形式输出到separated_colors文件夹
4、进入eda，根据自己期望的效果，把一个个蒙板叠到不同层，预期效果如下：
    ·全透光——顶层阻焊层+底层阻焊层
    ·半透明，不透光——顶层阻焊层
    ·浅阻焊色——顶层
    ·深阻焊色——无任何图片
    ·焊盘色（金/银）——顶层+顶层阻焊层（开窗）
    ·不透明白色——顶层丝印层

HINTS:
·不建议将黑色设置为“真全黑”，不然导入EDA后会识别不到图形
·先专注于取色（调hexColors），再专注于在PCB板上配色，一位优秀的画师往往需要先调出一套好看的色彩方案

Improvements:
·向量运算，秒出结果
·代码部分无需考虑繁杂细节，只需专注于创意发挥
